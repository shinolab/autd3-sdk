`FociStm` / `PatternStm` は AUTD3 デバイス内部でループされるため, ソフトウェアループは必要なく, 送信以降自動的に無限ループする.
また, タイミングは AUTD3 デバイス内蔵のタイマーで制御されるため, 精度が高く, 解像度も最高で 25 µs 単位で指定できる.
ただし, このタイマーの制約上, [出力不可能な周波数](/autd3-sdk/api/basics/sampling-config/#制約)が存在する. 

`FociStm` / `PatternStm` の切り替えタイミングは, 1 ループの周波数, または周期で指定する. 
すなわち, この場合は 2 つの焦点で 0.5 Hz (2 s) となるので, 各焦点は 1 s 出力される.
なお, [`SamplingConfig`](/autd3-sdk/api/basics/sampling-config/) を使用することで, 1 ループあたりではなく, サンプリング周波数, すなわち 1 焦点あたりの周波数や周期を指定することもできる. 

焦点に限らず任意の Pattern を切り替えたい場合は, `FociStm` の代わりに[`PatternStm`](/autd3-sdk/api/command/pattern-stm/) を使う.
