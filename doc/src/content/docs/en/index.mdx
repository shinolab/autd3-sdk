## What is AUTD3 SDK

AUTD3 SDK is a set of client libraries for controlling the **AUTD3** ultrasound phased array.
It is a next-generation, **incompatible** rewrite of the previous autd3 series, covering focus
generation, amplitude modulation, spatio-temporal modulation (STM) and hologram optimization
from Rust.
