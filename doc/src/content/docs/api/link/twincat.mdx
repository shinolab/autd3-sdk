`TwinCATLink` は, Windows 上の [TwinCAT](https://www.beckhoff.com/) を EtherCAT マスタとして利用する Link 実装. 
`autd3-rs-link-twincat` クレートで提供される.
TwinCAT がセットアップ済みの Windows 環境で使用する.
TwinCAT のセットアップ手順は [TwinCAT のインストール](/autd3-sdk/getting-started/twincat/) を参照すること.

ローカルの TwinCAT に接続する構成と, ADS 経由でリモートの TwinCAT サーバへ接続する構成がある.

## API

<CodeTabs rust={excerpt(sample, "api")} python={excerpt(pySample, "api")} />

| 引数         | 型         | 説明                                       |
| ------------ | ---------- | ------------------------------------------ |
| `addr`       | `IpAddr`   | リモート TwinCAT サーバの IP アドレス      |
| `ams_net_id` | `AmsNetId` | リモート TwinCAT サーバの AMS Net ID       |

タイムアウトを指定するには`*_with_timeouts`を使用する.
デフォルトはタイムアウトなし.
