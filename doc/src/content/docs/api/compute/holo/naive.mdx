Naive 法は, 目標音圧ベクトルに逆伝搬をかけるアルゴリズム. 
反復を行わず高速だが, 焦点間の干渉補正は行われないため, 弱い音場が生成される可能性がある. 

## API

<CodeTabs rust={excerpt(sample, "api")} python={excerpt(pySample, "api")} />

| 引数         | 型                     | 説明                      |
| ------------ | ---------------------- | ------------------------- |
| `geometry`   | `&Geometry`            | デバイス配置              |
| `foci`       | `&[ControlPoint]`      | 焦点列 (位置と目標振幅)   |
| `wavelength` | `Length`               | 波長                      |
| `option`     | `&NaiveOption`         | 追加オプション (下記参照) |
| `out`        | `&mut [[Emission; _]]` | 出力バッファ              |

共通要素 (`ControlPoint` / `TransducerMask` / `LinAlgBackend` など) は [Holo 概要](/autd3-sdk/api/compute/holo/) を参照. 

## NaiveOption

<CodeTabs rust={excerpt(sample, "option")} python={excerpt(pySample, "option")} />

| フィールド    | 型                   | デフォルト                    |
|---------------|----------------------|---------------------------|
| `constraint`  | `EmissionConstraint` | `Clamp(MIN, MAX)`         |
| `directivity` | `Directivity`        | `Sphere`                  |
| `backend`     | `impl LinAlgBackend` | `NalgebraBackend`         |
| `mask`        | `TransducerMask`     | `AllEnabled`              |

各フィールドの意味は [Holo 概要](/autd3-sdk/api/compute/holo/) を参照.
