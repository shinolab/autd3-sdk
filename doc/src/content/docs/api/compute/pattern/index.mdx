`autd3_rs_pattern` crateが, 単一焦点・平面波・ベッセルビームなどの代表的な音場計算ユーティリティを提供する.

## 関数一覧

- [focus](/autd3-sdk/api/compute/pattern/focus/) — 単一焦点
- [plane](/autd3-sdk/api/compute/pattern/plane/) — 平面波
- [bessel](/autd3-sdk/api/compute/pattern/bessel/) — ベッセルビーム
- [uniform](/autd3-sdk/api/compute/pattern/uniform/) — 全振動子一様
- [null](/autd3-sdk/api/compute/pattern/null/) — 無出力

各関数には, デバイス1台分のみを計算する `*_device` 版と, 振動子1素子分のみを計算する `*_transducer` 版もある (例: `focus_device` / `focus_transducer`).

## 波長

位相計算には波長 <Math tex="\lambda = c / f" /> が必要にになる.
音速 <Math tex="c" /> から `autd3_rs_pattern::wavelength(sound_speed)` で波長を求められる. 
音速 340 m/s (室温約15°の空気中での音速) では<Math tex="\lambda \approx 8.5\,\mathrm{mm}" />.
