`SetPulseWidthTable` は, 強度 (0–255) から PWM のパルス幅への変換テーブルを設定するコマンド.

## API

<CodeTabs rust={excerpt(sample, "api")} python={excerpt(pySample, "api")} />

| フィールド | 型                      | 説明                                                 |
| ---------- | ----------------------- | ---------------------------------------------------- |
| `table`    | `&[u16; PWE_TABLE_SIZE]` | 強度インデックス (0–255) ごとのパルス幅 (256 要素) |

`SetPulseWidthTable::default_table()` は arcsin 形状のデフォルトテーブルを返す.
これは出力音圧が強度に対して (理論的に) 線形になるようなテーブルである.
