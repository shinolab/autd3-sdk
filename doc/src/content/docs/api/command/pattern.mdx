## API

<CodeTabs>
<Fragment slot="rust">
<Code code={excerpt(sample, "api")} lang="rust" />

`Pattern::new` は書き込み先バンクを `PatternBank::B0` とする.
別バンクへ書く場合は `Pattern::with_bank(bank, emissions)` を使う.
</Fragment>
<Fragment slot="python">
<Code code={excerpt(pySample, "api")} lang="python" />

`Pattern` は書き込み先バンクをデフォルトで `PatternBank.B0` とする.
別バンクを指定したい場合は `bank` 引数を渡す.
</Fragment>
</CodeTabs>

| フィールド        | 型                               | デフォルト                      | 説明                             |
| ----------------- | -------------------------------- | --------------------------- | -------------------------------- |
| `bank`            | `PatternBank`                    | `PatternBank::B0`           | 書き込み先・再生対象のバンク     |
| `emissions`       | `&[Vec<Emission>]` | —                           | デバイスごとの emission パターン |

patternの計算には幾つかの[ヘルパ関数](/autd3-sdk/api/compute/pattern/)が用意されている.
