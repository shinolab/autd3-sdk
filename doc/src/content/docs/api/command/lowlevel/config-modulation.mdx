`ConfigModulation` は, Modulation バンクの再生パラメータを設定する低レベルコマンド.

## API

<CodeTabs rust={excerpt(sample, "config")} python={excerpt(pySample, "config")} />

| フィールド | 型               | 説明                                              |
| ---------- | ---------------- | ------------------------------------------------- |
| `bank`     | `ModulationBank` | 対象バンク                                        |
| `config`   | `SamplingConfig` | 再生サンプリング周波数                            |
| `size`     | `usize`          | サンプル数 (`data.len()` と一致させる)            |
| `loop_behavior` | `LoopBehavior` | 再生ループ回数 (`Infinite` / `Finite`)         |

`loop_behavior` と遷移の反映タイミングは [ChangeModulationBank](/autd3-sdk/api/command/lowlevel/change-modulation-bank/) を参照.

## Example

`ConfigModulation` は `WriteModulationBuffer` で波形を書き込んだ後, `ChangeModulationBank` で再生を切り替える流れの中で使う.
