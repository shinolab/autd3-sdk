`WriteModulationBuffer` は, 指定したバンク・オフセットへ AM 波形 (`&[u8]`) を書き込む低レベルコマンド.

## API

<CodeTabs rust={excerpt(sample, "write")} python={excerpt(pySample, "write")} />

| フィールド | 型               | 説明                             |
| ---------- | ---------------- | -------------------------------- |
| `bank`     | `ModulationBank` | 書き込み先バンク                 |
| `offset`   | `usize`          | バンク内の書き込み開始オフセット |
| `data`     | `&[u8]`          | AM 波形サンプル列                |

## Example

`WriteModulationBuffer` → `ConfigModulation` → `ChangeModulationBank` で AM 波形を再生する例 (高レベルの [Modulation](/autd3-sdk/api/command/modulation/) と等価).
