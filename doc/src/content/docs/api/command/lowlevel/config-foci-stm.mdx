`ConfigFociStm` は, Foci STM バンクの再生パラメータを設定する低レベルコマンド.

## API

<CodeTabs rust={excerpt(sample, "config")} python={excerpt(pySample, "config")} />

| フィールド      | 型               | 説明                                               |
| --------------- | ---------------- | -------------------------------------------------- |
| `bank`          | `PatternBank`    | 対象バンク                                         |
| `config`        | `SamplingConfig` | 再生サンプリング周波数                             |
| `size`          | `usize`          | STM サンプル数                                     |
| `num_foci`      | `u8`             | 1 サンプルあたりの焦点数 (`ControlPoints<N>` の N) |
| `sound_speed`   | `Velocity`       | 焦点計算に使う音速                                 |
| `loop_behavior` | `LoopBehavior`   | 再生ループ回数 (`Infinite` / `Finite`)             |

`loop_behavior` と遷移の反映タイミングは [ChangePatternBank](/autd3-sdk/api/command/lowlevel/change-pattern-bank/) を参照.

## Example

`ConfigFociStm` は `WriteFociBuffer` で焦点列を書き込んだ後, `ChangePatternBank` で再生を切り替える流れの中で使う.
