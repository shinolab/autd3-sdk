`Modulation` は, AM 波形データ (`&[u8]`) をデバイスへ送るコマンド.

## API

`Modulation::new` は書き込み先バンクを `ModulationBank::B0`, ループを `Infinite`, 遷移を `Immediate` とする.
別バンクへ書く場合は `Modulation::with_bank(bank, config, data)` を使う.
ループ回数や遷移タイミングも変えたい場合は, フィールドを直接指定して構築する.

`Modulation` は書き込み先バンクをデフォルトで `ModulationBank.B0`, ループを `Infinite`, 遷移を `Immediate` とする.
別バンク・ループ回数・遷移タイミングを指定したい場合は, `bank` / `loop_behavior` / `transition_mode` 引数を渡す.
</Fragment>
</CodeTabs>

| フィールド        | 型               | デフォルト                  | 説明                         |
| ----------------- | ---------------- | --------------------------- | ---------------------------- |
| `bank`            | `ModulationBank` | `ModulationBank::B0`        | 書き込み先・再生対象のバンク |
| `config`          | `SamplingConfig` | —                           | 再生サンプリング周波数       |
| `data`            | `&[u8]`          | —                           | AM 波形サンプル列            |
| `loop_behavior`   | `LoopBehavior`   | `LoopBehavior::Infinite`    | 再生ループ回数               |
| `transition_mode` | `TransitionMode` | `TransitionMode::Immediate` | バンク切替時の遷移挙動       |

`config` は [`SamplingConfig`](/autd3-sdk/api/basics/sampling-config/) で指定する.
`data` は `modulation_buffer()` で確保したバッファへ書き込む.
`loop_behavior` / `transition_mode` の挙動は [ChangeModulationBank](/autd3-sdk/api/command/lowlevel/change-modulation-bank/) を参照.

modulation の計算には幾つかの[ヘルパ関数](/autd3-sdk/api/compute/modulation/)が用意されている.
