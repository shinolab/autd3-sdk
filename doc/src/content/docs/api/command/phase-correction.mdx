`SetPhaseCorrection` は, 各振動子へ加える位相オフセットを設定するコマンド.
ここで設定したオフセットは以降のすべての出力位相へ加算される.

<CodeTabs rust={excerpt(sample, "api")} python={excerpt(pySample, "api")} />

| フィールド | 型              | 説明                                                                                     |
| ---------- | --------------- | ---------------------------------------------------------------------------------------- |
| `phases`   | `&[Vec<Phase>]` | デバイスごとの位相オフセット表. |
